# SimpleGym - Full Stack App

Este repositorio contiene la solución completa de SimpleGym, dividida en un cliente de React y una API en Node.js.

## Estructura del Proyecto

* [**Frontend (React + Vite)**](./frontend/README.md): Interfaz de usuario, gestión de entrenamiento y catálogo.
* [**Backend (Node.js + Express)**](./backend/README.md): API REST y persistencia de datos en JSON.

## Instalación Rápida

Para ejecutar el proyecto completo, es necesario iniciar ambos servicios:

1. **Backend**: Ir a `/backend`, ejecutar `npm install` y luego `npm run dev`.
2. **Frontend**: Ir a `/frontend`, ejecutar `npm install` y luego `npm run dev`.
